# AASTU Event Hub

This repository is organized into separate frontend and backend folders for a clean, professional structure.

## Folder structure

- `frontend/` - the React + Vite application, including source code, build config, and public assets.
- `backend/` - reserved for future server-side code. There is no backend implementation yet.

## Development

From the repository root, run:

```bash
npm run dev
```

This starts the frontend dev server from `frontend/`.

## Backend

Add backend code under `backend/` if you later implement APIs, auth, or database services.
